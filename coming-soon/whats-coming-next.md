# Windows Support

Nanosandbox is coming to Windows. By leveraging WHPX (Windows Hypervisor Platform), we will bring the same lightweight microVM experience to Windows users.

## What This Means

- The same `nanosb` CLI and TUI you use on macOS and Linux will work on Windows
- WHPX provides native hypervisor-level isolation without requiring third-party virtualization software
- Cross-platform teams can standardize on a single sandboxing tool across macOS, Linux, and Windows

## Why It Matters

Teams that span multiple operating systems should not need different tooling per platform. Windows support means every developer on the team gets the same interface, the same commands, and the same guarantees — regardless of their OS.
